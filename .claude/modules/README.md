# Mục lục các Module

Thư mục `modules/` chứa các mô tả chi tiết cho từng module của dự án.

Mỗi module nên có một file markdown riêng, bao gồm:
- Kiến trúc chi tiết.
- Luồng hoạt động chính.
- API, mô hình dữ liệu và các dependency quan trọng (nếu có).
- Các quyết định quan trọng và lý do lựa chọn.
- Trạng thái hiện tại và các công việc còn lại.
