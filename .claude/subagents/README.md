# Thư mục Context Subagents

Thư mục này **không** phải là một phần của mã nguồn dự án e‑commerce. Nó lưu trữ các file markdown mô tả ngữ cảnh của các AI sub‑agent (ví dụ: research, analysis) được trợ lý sử dụng trong các phiên làm việc. Những file này giúp AI tách biệt lý luận khỏi logic dự án và tránh làm nhiễu codebase.

- **Không** xem nội dung này như các file nguồn cho ứng dụng.
- Chúng chỉ phục vụ cho việc quản lý nội bộ của trợ lý.
- Các module dự án vẫn nằm trong `modules/`.
